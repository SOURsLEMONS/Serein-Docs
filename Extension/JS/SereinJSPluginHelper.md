## 使用方法

- 建议使用 Visual Studio Code 编写插件 下载：<https://code.visualstudio.com/>

1. 将此文件复制到插件的同文件夹下

2. 在你的插件第一行加上下面这一行

    ```js
    /// <reference path="SereinJSPluginHelper.js"/>
    ```

3. 然后你就可以快乐地写插件了，这时候就可以自动补全和显示函数参数了！！

>[!TIP]发布时**不需要**带上此插件
